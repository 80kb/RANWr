# RANWr 
 RANWr (RANWr Ain't NintendoWare) is set of editors for various Wii file formats.
 
## Current Tools
| Tool   | Status   |  Formats | Can Open | Can Save |
|--------|----------|----------|----------|----------|
| AnimSoundMaker| In Works | .brasd .rasd .rasp| Yes      | No     |
| SoundMaker | Planned | - | - | - |
