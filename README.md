# RANWr 
 RANWr (RANWr Ain't NintendoWare) is set of editors for various Wii file formats.
 
# Current Tools
## AnimSoundMaker: Early Alpha
#### Supported File Formats
| Format | Can Open | Can Save |
|--------|----------|----------|
| .rasd  | Yes      | Yes      |
| .brasd | Yes      | Yes      |
| .rasp  | Yes       | No       |

## ArchiveExplorer: In Progress
